# 🎮 3D Adventure Game - IMR HAND

A modern, visually striking 3D adventure game website with immersive gameplay and clean, minimal UI design. Built with cutting-edge web technologies and advanced programming concepts.

## ✨ Features

### 🎨 **Visual Design**
- **Immersive Splash Screen** - Particle effects, smooth animations, matching "IMR HAND" design aesthetic
- **Minimal UI** - Glass-morphism HUD with essential information only
- **Professional Color Palette** - Orange (#FF5722), Dark (#1a1a1a), Peach accent (#FFB6A3)
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices

### 🎮 **Gameplay**
- **Controllable Character** - Walk (WASD), Jump (SPACE), Interact (E)
- **Physics System** - Realistic gravity, smooth movement, ground detection
- **Collectibles** - Golden items to collect, health boosts, item counter
- **Interactive World** - Terrain, trees, rocks, dynamic lighting
- **Third-Person Camera** - Smooth following camera with cinematic feel

### ⚡ **Technical Excellence**
- **High-Performance Rendering** - WebGL acceleration, optimized shadow mapping
- **Modular Architecture** - Clean separation of concerns, easy to extend
- **Advanced Animations** - Smooth transitions, particle effects, floating objects
- **Responsive Performance** - 60+ FPS, mobile-optimized rendering
- **Real-time Monitoring** - FPS counter and debug information

## 🛠️ **Tech Stack**

- **3D Graphics** - Three.js + WebGL
- **Frontend** - Vanilla JavaScript (ES6 modules)
- **Styling** - Modern CSS with animations
- **Performance** - GPU acceleration, optimized rendering pipeline
- **Physics** - Custom physics system with gravity and collision detection

## 📁 **Project Structure**

```
├── index.html              # Main entry point
├── main.css                # Styling and animations
├── package.json            # Dependencies
└── src/
    ├── main.js             # App orchestrator
    ├── SplashScreen.js      # Splash screen with particles
    ├── Game.js              # Core game engine
    ├── Renderer.js          # WebGL renderer setup
    ├── Scene.js             # 3D environment
    ├── Player.js            # Player entity & physics
    ├── InputManager.js       # Input handling
    └── CameraController.js   # Camera system
```

## 🎮 **Controls**

| Key | Action |
|-----|--------|
| **WASD** | Move character |
| **SPACE** | Jump |
| **E** | Interact |
| **ESC** | Pause/Resume |
| **Click** | Skip splash screen |

## 🚀 **Getting Started**

### Installation

```bash
# Clone the repository
git clone https://github.com/DangerousEra/3D-adventure-game.git
cd 3D-adventure-game

# Install dependencies
npm install
```

### Running Locally

```bash
# Start development server
npm run dev

# Open browser to http://localhost:5173
```

### Building for Production

```bash
npm run build
```

## 🎯 **Gameplay Guide**

1. **Splash Screen** - Watch the immersive intro with particle effects
2. **Enter Game** - Click or press SPACE to start
3. **Explore** - Walk around the 3D world using WASD
4. **Collect Items** - Walk over golden collectibles to gain health
5. **Interact** - Press E near interactive elements
6. **Pause** - Press ESC to pause and resume
